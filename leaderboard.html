Leaderboard

This page contains the assignment leaderboard. The leaderboard is updated as follows. Every five minutes, on the five-minute mark, every assignment not yet past its due date is downloaded according the base URL and assignment number you turned in with Assignment 0. The output is rescored if it changed.

Legend

A value of -1 indicates that the assignment file was found but contained invalid content.

The light coral line is the default system; it is what you should get if you run the code that was provided for you. The row highlighted with pale golden rod is the baseline you should strive to beat. Typically, this line depicts the minimal effort required, which will earn you a B (7 points) on the project. The best system will earn an additional 6 points, the second best will earn an additional 5 points, and so on.
